# Automation Lead Capture Landing Page

A simple, conversion-focused landing page designed to capture leads from businesses struggling with repetitive tasks such as spreadsheets, email handling, and data processing.

Users submit their task, and receive a free custom automation demo (via Loom) showing how the workflow could be automated.

## 📌 Project Overview

This project is a lead generation landing page built for offering **automation services to businesses**.

The goal is to:
- Capture real business problems (manual repetitive tasks)
- Demonstrate how automation can solve them
- Convert visitors into leads through a simple form

Instead of selling a product, the page offers a **free automation demo** tailored to the user's specific workflow.

---

## 🎯 Problem It Solves

Many small businesses still spend hours on:
- Spreadsheet cleaning and formatting
- Copying and moving data between tools
- Manual email processing
- Lead organization and updates

This landing page provides a way to:
> Replace repetitive work with simple automation scripts and workflows.

---

## ⚙️ How It Works

1. User lands on the page
2. They describe a repetitive task they struggle with
3. They submit the form
4. The request is received via email
5. A custom automation demo is created (Loom video)
6. User receives a practical solution for their workflow
